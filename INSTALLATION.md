# Installation Guide

1. **Clone the repository:**
   ```bash
   git clone https://github.com/essamamdani/open-crypto-icons.git
   cd open-crypto-icons
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run locally:**
   ```bash
   npm run dev
   ```
4. **Fetch Icons:**
   Run the python script to update icons.
   ```bash
   python3 fetch_icons.py
   ```
